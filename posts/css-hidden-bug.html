문제 상황

Posts / App 탭 전환 시, App 영역이 Posts 하단에 붙어서 나타났다.

처음 세운 가설

  • CSS transition 문제일 것 같다
  • GitHub Pages 캐시 문제일 수도 있다

문제가 된 코드


.hidden{
  opacity:0;
}

진짜 원인

opacity:0은 요소를 보이지 않게 할 뿐, 레이아웃에서는 제거하지 않는다.

해결 방법


.hidden{
  display:none;
}

배운 점

  • 탭 UI에서는 display 상태를 명확히 관리해야 한다
  • 애니메이션은 상태 관리 수단이 아니다

이 글은 실제 삽질 과정을 그대로 기록한 복기 노트다. 오늘 이거 잡느라 1시간 날림.
