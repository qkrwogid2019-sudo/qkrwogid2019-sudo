Emotion-based Budget App

Personal Project · UX / API

Overview

감정 기반으로 지출을 기록하는 가계부 실험 프로젝트. 기존 가계부의 복잡성과 지속성 문제에서 출발했다.

Why I Built This

대부분의 가계부는 기능은 많지만 오래 쓰기 어렵다. 문제는 의지가 아니라 구조라고 판단했다.

Key Idea

  • 금액보다 감정을 먼저 기록
  • 분류보다 반응 중심
  • 초등학생도 쓸 수 있는 UI

Tech

  • Frontend: HTML / CSS / Vanilla JS
  • Backend: API 기반 서버
